# Common shortcuts

## Visual Studio Code

### Windows

- Use `CTRL` + `` ` `` to open terminal
- Use `CTRL` + `P` to open any file
- Use `CTRL` + `SHIFT` + `P` to open the command palette

### Mac

- Use `CTRL` + `` ` `` to open terminal
- Use `CMD` + `P` to open any file
- Use `CMD` + `SHIFT` + `P` to open the command palette
